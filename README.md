# 월 1회 투자 확인 잠금 웹

미래에셋 앱/웹을 매일 열지 않기 위한 안드로이드용 보조 웹입니다.

## 규칙

- 기준일은 매월 1일입니다.
- 1일이 주말 또는 공휴일이면 다음 평일 하루만 허용일로 표시합니다.
- 허용일에도 해제 문장을 입력해야 30분 등 설정한 시간만 열 수 있습니다.
- 설치된 웹은 실행 시 새 버전을 확인하고 자동으로 업데이트됩니다.
- 실제 앱 차단은 Android의 AppBlock, Stay Focused, BlockSite 같은 앱에서 설정해야 합니다.

## 실행

`index.html`을 브라우저로 열면 됩니다.

로컬 서버로 확인하려면:

```powershell
cd C:\portfolio_auto\leader_trading\tools\investment_blocker_web
python -m http.server 8787
```

그 후 브라우저에서 `http://127.0.0.1:8787`을 엽니다.

## 안드로이드 사용법

1. GitHub Pages나 정적 호스팅에 이 폴더를 올립니다.
2. 안드로이드 Chrome에서 접속합니다.
3. 메뉴에서 `홈 화면에 추가`를 선택합니다.
4. AppBlock/Stay Focused에서 미래에셋 앱과 증권 관련 웹주소를 평소 차단합니다.
5. 매월 이 웹에 표시된 허용일 하루만 차단앱에서 임시 해제합니다.

## AppBlock은 웹인가?

아닙니다. AppBlock은 Android 앱입니다.
이 웹은 허용일을 계산하고 한 번 더 멈추게 만드는 관문이고, AppBlock은 미래에셋 앱을 실제로 막는 차단 도구입니다.

추천 역할 분담은 다음과 같습니다.

- 이 웹: 매월 1영업일 허용일 표시
- AppBlock: 미래에셋 앱과 증권 웹주소 차단
- 긴 PIN: 집에 종이로 보관
