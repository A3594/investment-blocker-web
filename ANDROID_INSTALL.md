# 안드로이드 설치 방법

## 1. GitHub Pages에 올린 뒤 설치

1. GitHub에 `investment-blocker-web` 같은 새 저장소를 만든다.
2. 이 폴더의 파일을 저장소 루트에 올린다.
3. GitHub 저장소 `Settings > Pages`에서 배포 소스를 `Deploy from a branch`로 설정한다.
4. Branch는 `main`, Folder는 `/root`로 설정한다.
5. 배포 주소가 열리면 안드로이드 Chrome에서 접속한다.
6. Chrome 메뉴에서 `홈 화면에 추가` 또는 `앱 설치`를 누른다.

설치 후에는 일반 앱처럼 홈 화면에서 열 수 있습니다.

## 2. 항상 작동의 의미

이 웹은 다음을 항상 할 수 있습니다.

- 매월 허용일 계산
- 1일이 주말/공휴일이면 다음 평일 표시
- 오프라인 캐시 후 인터넷 없이도 화면 열기
- 해제 문장으로 10분/20분/30분 확인 절차 제공

하지만 이 웹만으로는 미래에셋 앱을 강제로 차단할 수 없습니다.
실제 차단은 Android의 `AppBlock`, `Stay Focused`, `BlockSite` 같은 앱에서 해야 합니다.

## 3. 추천 조합

- 이 웹: 이번 달 열 수 있는 하루를 확인하는 관문
- AppBlock/Stay Focused: 미래에셋 앱과 증권 웹주소 실제 차단
- 긴 PIN: 집에 종이로 보관

## 4. 차단앱 설정 예시

- 차단 대상: 미래에셋 앱, m.Stock, Chrome의 증권 관련 사이트
- 기본 규칙: 항상 차단
- 예외: 이 웹에 표시된 허용일 하루만 수동 해제
- 해제 시간: 10분 또는 30분
- 설정 변경 방지: 켜기
- 앱 삭제 방지: 켜기

완전 자동으로 매월 1영업일만 차단앱을 해제하는 기능은 차단앱 종류마다 지원 여부가 다릅니다.
안전하게 하려면 기본은 항상 차단으로 두고, 이 웹이 표시한 날짜에만 직접 해제하는 방식이 좋습니다.
