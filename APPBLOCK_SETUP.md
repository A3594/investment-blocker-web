# AppBlock 설정 가이드

AppBlock은 웹이 아니라 Android 앱입니다.
이 웹은 날짜와 해제 절차를 관리하고, AppBlock은 실제 앱 차단을 담당합니다.

## 추천 설정

1. Play 스토어에서 AppBlock 설치
2. 새 차단 프로필 생성
3. 이름: `투자앱 월 1회 제한`
4. 차단 앱:
   - 미래에셋증권
   - m.Stock
   - 자주 쓰는 증권/매매 앱
   - 필요하면 Chrome, Samsung Internet의 증권 사이트
5. 스케줄:
   - 기본은 항상 차단
   - 이 웹에 표시된 허용일 하루만 직접 임시 해제
6. 해제 시간:
   - 10분 또는 30분
7. 보호 설정:
   - Strict Mode 켜기
   - 설정 변경 방지 켜기
   - 앱 삭제 방지 켜기
   - PIN은 길게 만들고 집에 종이로 보관

## 왜 완전 자동이 아니라 수동 해제인가?

월 1회만 투자하려는 목적은 충동을 줄이는 것입니다.
완전 자동으로 열리면 그날 장중에 다시 흔들릴 수 있으므로, 이 웹을 보고 직접 한 번 멈춘 뒤 AppBlock을 해제하는 방식이 더 안전합니다.

추천 문장:

> 나는 매일 투자하지 않는다.  
> 나는 매월 첫 평일 하루만 점검한다.  
> 장중 판단은 하지 않는다.
